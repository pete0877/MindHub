Tracking Your Work in the Results window

The Results window is an interface for initiating viewing and opens automatically for commands that generate testing results

This tabbed interface is the work area for:

  • Search -- Shows matches from Extended Find and Replace operations.
  • Validation -- Displays code validator messages.
  • Links -- Lists links in current document for testing.
  • Thumbnails -- Displays all recognized image files in the current directory.
  • Deployment -- Presents information on project file uploads.

Printing results

Right-click in the Search, Validation, Links, or Deployment tab and select Browse to create an HTML document containing the current results in that tab. You can then save and print the document for review.
