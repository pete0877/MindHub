Customizing the Workspace

A development tool needs to provide a productive interface for its users, and the best way to do that is to give users the greatest possible flexibility to set up the interface in a way that works for them. Here are some options:

  • Press Ctrl + F12 to toggle the Full Screen view.
  • Press F9 or open the View menu to toggle the Resources window display.
  • To resize the Resources window, drag the side border and the divider between the top and bottom panes.
  • The Resources window can be moved and docked in a variety of locations by dragging the grab bar at the top.
  • Resource tabs can be floated as individual windows by clicking the tab and dragging it to a new location. Resource tabs can be docked together in any combination by dragging them to the same location.
  • Floating Resource windows can be made to float on top of the main Studio window by right-clicking the title bar and choosing Always on top.
  • The order of Resource tabs in any window can be changed by right-clicking the Resource tab and dragging.
  • Use the General pane of the Settings dialog box (F8) to control the display of Resource tabs in a window.
  • Right-click in the file list to set file display options.
  • Drag toolbars to any location as floating windows or dock them vertically on the workspace borders.
  • Right-click in any toolbar to set display options and to organize and customize toolbars.
  • Open the Options > Customize dialog box (Shift + F8) to change toolbar layout, keyboard shortcuts, and to add shortcut keys for code snippets.
  • Open the Options > Settings dialog box (F8) to explore a wide range of interface settings and development options.
