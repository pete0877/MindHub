Working in the Resources Area

The tabs on this window control display for these Resources:

  • Universal Files -- Provides access to all of your files, including those on local drives, network drives, and remote servers. Provides functionality to add and remove remote servers. The drop-down list allows you to pick a drive or server while the bottom two panes display the directories and files for the drive or server you selected. Double-click a file in the file list to open it. See "Managing Files" for more information.
  • Projects -- Provides functionality to create and manage projects. The drop-down list allows you to pick a recent project. The bottom two panes display the folders and files for the selected project. Right-click in the panes to open and create projects and to add projects to your version source control system. See Chapter 10, "Using Projects for Site Management" for more information.
  • Site View -- Displays a graphic representation of links in the current document. See "Using Site View to Check Page Links".
  • Snippets -- Stores code blocks for re-use. Snippets can be shared with other users. See "Saving Code Blocks as Snippets" for more information.
  • Help -- Displays Studio documentation and other online resources. You can customize Studio's help system by editing existing help files and adding new files. See Chapter 4, "Getting the Most from the Help System" for more information.
  • Tag Inspector -- The top pane displays the Tag Tree, a customizable view of the document hierarchy. The bottom pane displays the Tag Inspector, which gives you an interactive "property sheets" mode for editing tags. See "Tag Tree and Tag Inspector" for more information.
