Chapter 7 :  Accessing ColdFusion Data Sources



Connecting to Data Sources

Data sources registered in the ColdFusion Administrator are automatically accessible from Studio's Database tab. You can also add remote ColdFusion servers (RDS) to connect to their data sources.

To add a remote server:
  1. Open the Database tab and select Add RDS server from the dropdown list in the top pane.

  2. Complete the Configure RDS Server dialog.

    See "Connecting to Remote Servers" for detailed instructions on configuring an RDS server.
