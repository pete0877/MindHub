Using ColdFusion Studio
Chapter 4 :  Getting the Most from the Help System



Using Help References

The Help tab contains the printed documentation in HTML format plus a number of additional online references. The help system is an extensible resource for learning about Studio's features and Web site development.

Click the Help tab in the Resources pane to view the Help References tree. Click the plus sign (+) next to a book icon to expand the document list and double-click a help document to open it.

Changing the help display location

By default, help documents display in the Browse tab of the Editor window. To view them in a pane below the Help tree, click the Browse help in separate panel button on the Help toolbar. You can then adjust pane size by dragging it's borders.

To view a help document in an external browser. click the Open in external browser button in the pane below the Help tree.

Bookmarking help documents

You can get quick access to documents by bookmarking them. Bookmarks can be assigned to documents in the Help tree and to HTML documents on local and network drives. The Bookmark icon is located on the Help toolbar.

To bookmark a document in the Help tree:
  1. Right-click on a document title.
  2. Select Add bookmark from the menu.

    The document's name is added to the Bookmark list, accessed by clicking the down arrow next to the Bookmark icon.

To bookmark an external document:
  1. Click the Bookmark icon to open the Help Bookmarks dialog box.

  2. Click the Add button to see the Add Booksmarks dialog box.
  3. Use the Browse button, type in the path or URL, or paste in a URL from a browser.
  4. Click OK to add the bookmark to the Help Bookmarks dialog box.
  5. Click Close to save your changes and close the Help Bookmarks dialog box.
To change the title of a bookmark:
  1. Click the down arrow next to the bookmark icon to open the bookmark list and choose Organize Bookmarks from the menu.
  2. Click a title in the bookmarks list and then click Rename to highlight the title text (alternatively, you can double-click a bookmark to highlight its title).
  3. Type the new title and press the Enter key to save the new name.
  4. Click Close to save your changes and close the Help Bookmarks dialog box.
To open a bookmarked document:
  1. Click the down arrow next to the Bookmark icon to open the bookmark list.

  2. Select a document from the list.

    You see the document in the Browse tab of the Editor window.

To delete a bookmark:
  1. Click the Bookmark icon to open the Help Bookmarks dialog box.
  2. Select the bookmark and click Remove.
  3. Click Close to save your changes and close the Help Bookmarks dialog box.
