Searching Help References

Studio's help text search feature lets you do everything from a simple title search to advanced searches using complex criteria.

To run a search:
  1. Click the Search button on the Help toolbar.

    You see the Search Help References dialog box.

    The first time you open the search dialog box, you will be prompted to index the Help References. We recommend doing this, as it greatly improves search engine performance.

  2. Enter a word or phrase in the Enter the word(s) to find box.

    To read about using search operators, click the Search Tips button. For a detailed description of the help system's powerful search options, see the Using advanced search operators section below.

  3. To limit the references searched, click the Search only in selected references option and select one or more references by clicking in the box next to the reference name.
  4. Click Search.

    The search results display in the Help References window.

  5. Double-click a document in the list to open it.

The last results list is saved until a new search is run, so you can return to the results list by clicking the Results button on the Help toolbar.

Note You can rerun a recent search by selecting it keywords from the Enter the word(s) to find drop-down list in the Search Help References dialog box. Choose an entry and then click Search.

Each time Studio loads, it checks the Help References tree for changes and automatically re-indexes the documents if needed.

Using advanced search operators

You can use a variety of advanced search operators to improve your searching in the Help references. For information on using advanced search operators, click the Search Tips button in the Search Help References dialog box. You can print this reference by using Windows Explorer to open the search_tips.pdffile in the /Help/Allaire Support folder in your Studio installation directory.
