Using Pop-up Help in the Editor

Studio's Editor window offers a variety of online help to aid you in completing tags and functions with the proper syntax.

Tag Tips

Tag tips give you a popup display of the syntax for the current tag. Place the cursor inside a tag and press F2 to open a Tag Tip. Press Esc to close it.

Tag insight

Tag insight lets you select attributes as you type in the tag code. See "Getting Help Entering Content" for details.

Function Insight

Function insight, similar to Tag insight, lets you select function arguments as you type in the function code. See "Getting Help Entering Content" for details.
