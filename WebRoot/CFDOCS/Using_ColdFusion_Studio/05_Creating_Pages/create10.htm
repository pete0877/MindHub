Inserting Tags from the QuickBar

When you click on a QuickBar icon, it either inserts the tag directly into the document or opens a dialog box for tags that require attributes. Each dialog box is tag-specific; it only displays supported attributes.

You can easily customize the QuickBar display and organize toolbuttons.

To move a QuickBar tab:
  1. Click and hold on the drag bar -- the double lines located at the left end of the tab -- of the tab you want to move.
  2. Drag the toolbar to the new position. To place the toolbar on a workspace border, drag it to the edge of the border until the toolbar border changes to thin lines.
  3. Release the mouse button when the toolbar is in its new location.
  4. To return a toolbar to its previous location, double-click on its title bar.
To organize QuickBar tabs:
  1. Right-click in the QuickBar and select the tabs you want displayed.
  2. Right-click in the QuickBar and select Organize QuickBar.
  3. Use the up and down arrows to set the tab order.
To customize QuickBar tabs:
  1. Right-click in the QuickBar and select Customize.
  2. Double-click on a toolbar name in the Visible Toolbars list.
  3. Drag and drop items from the Toolbuttons list onto the selected toolbar.
  4. Click the Add Separator button to add a vertical bar next to the last icon.
  5. Check the Show keyboard shortcuts in toolbutton hints option to display the key combination when you hover the mouse pointer over a toolbutton.
To add a toolbar:
  1. Click Add Toolbar in the Customize dialog box.
  2. Enter a name for the new toolbar and click OK to add the toolbar to the Toolbars list.
  3. Select the new toolbar from the list and add toolbuttons as needed.
  4. Use the Organize QuickBar dialog box to determine the placement of the toolbar on the QuickBar.
To add a custom toolbutton:
  1. Click Add Custom Button to open the Custom Toolbutton dialog box.
  2. Click the radio button to select an action for the toolbutton.
  3. Complete the entries for the selected action.
  4. Click OK to add the new toolbutton to the selected toolbar.

You can always return to the original QuickBar configuration by clicking the Reset to Defaults button.
