Using ColdFusion Studio

Chapter 5
Creating Pages

Studio is designed to support many different development tasks, from writing simple HTML pages to designing, testing, and deploying complex, dynamic Web sites.

This chapter describes some basic techniques and offers pointers to features and settings that can help you build Web pages in the most productive way.

If you need to gain a basic understanding of HTML and Web page creation, see Chapter 3, "Learning HTML and CSS".
