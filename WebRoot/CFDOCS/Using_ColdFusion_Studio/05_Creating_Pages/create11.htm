Using Keyboard Shortcuts

Studio offers keyboard shortcuts for many file management, editing, and debugging commands. You can also customize these shortcuts. Choose Options > Customize (Shift + F8) and click the Keyboard Shortcuts tab to see the current keyboard mappings and to edit them to suit your preferences.

A printable list of shortcut keys, in Adobe Acrobat format, is contained in the shortcut_keys.pdf file in the Allaire Support folder. You can locate the file with Windows Explorer.
