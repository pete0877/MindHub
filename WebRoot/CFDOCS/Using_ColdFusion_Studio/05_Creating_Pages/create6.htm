Printing Code, Browser Pages, and Help Topics

To print the code and text of the current document, select File > Print while in Edit, Browse, or Design modes. You can set print options for numbering, page headers, and color.

To print a page as it appears in the browser, use either of these options:

  • If you are using Internet Explorer as the internal browser, right-click on the document in Browse mode and select Print.
  • Open the document in your default external browser (F11) and use the browser's Print command.

To print a page in the Help References tab, use either of these options:

  • If you are using Internet Explorer as the internal browser, right-click in the help page and select Print.
  • Click the Open in external browser button on the Browser toolbar. You can then print the page from the browser.
