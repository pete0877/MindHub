Deploying to Multiple Servers

It is possible with the Deployment Wizard to deploy to multiple servers. It is assumed when you deploy to multiple servers that those servers are mirrored and contain the same directory structure. When you deploy to multiple servers, the project files are copied to identical locations on each server.
