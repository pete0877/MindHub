Chapter 10 :  Using Projects for Site Management



The Projects Tab

Projects are managed through the options you see on the Projects tab. To display the the project options, click the Projects tab at the bottom of the Resources window.

The following illustration shows the features on the Projects tab:

  • Recent Projects -- Display a list of recent projects that you can open.
  • Project -- Displays the name of the project and acts as the project tree root.
  • Project Folders -- Displays the folders that you created to hold the files in your project.
  • Resources -- Displays the files in your project, organized by file type.
  • Deployment Servers -- Displays a list of Web servers to which you can upload your project.
  • Deployment Scripts -- Displays a list of scripts that can be run in order to deploy the current project to its Web server(s).
  • Project Files -- Displays a list of files that exist in the current folder.
  • Open Project -- Allows you to open any project on your local or remote drives.
  • New Project -- Allows you to create a new, empty project.
  • Deploy Project: Opens the Deployment Wizard, which allows you to upload your project to the Web server.
    Note The Project Files list that is displayed in the bottom pane of the Projects tab can be treated very similarly to the file list that is displayed as part of the Universal Files tab. In other words, you can drag and drop files or open files for editing from this list.
