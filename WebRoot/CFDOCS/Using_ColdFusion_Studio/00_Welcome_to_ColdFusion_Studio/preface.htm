Using ColdFusion Studio

Welcome to ColdFusion Studio

ColdFusion Studio is a powerful and flexible tool for building, testing, and deploying Web content. It is also an easy-to-use editor with support for HTML, DHTML, VTML, XML, HDML, SMIL, and a variety of scripting languages.
