Using ColdFusion Studio

Chapter 3
Learning HTML and CSS

This tutorial is divided into two parts.

The first section takes you through the steps of creating a Web page with the basic elements of page formatting, images, hyperlinks, tables, and lists. The HTML tags used to build the example page are described along with the Studio options and commands for entering the code. The tutorial can be completed in about half an hour.

The second section shows you how to create the same page design using Cascading Style Sheets.
