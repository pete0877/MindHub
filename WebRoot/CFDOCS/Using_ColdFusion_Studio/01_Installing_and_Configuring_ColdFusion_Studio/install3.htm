Importing Settings from a Previous Version

The first time you run Studio, it looks for a previously installed version (Studio 4.0 and later) on your system. If a previous version is found, the Initial Configuration Wizard runs to let you import your customizations and other settings into the new version.

The wizard checks previous version information in the Windows Registry and the UserData folder and lists the settings it finds.

If you uninstall Studio or delete files from the UserData folder before installing the new version, Studio will not be able to detect those settings and they will not appear in the wizard.

The list of items you may be asked to choose from includes:

  • Remote servers
  • Custom toolbars and toolbuttons
  • Snippets and Snippet shortcuts
  • Special characters
  • Closing tags
  • Code templates
  • Keyboard shortcuts
  • Tag insight settings
  • Editor customizations
  • Window positioning settings are imported automatically without user intervention

If you want to run the wizard again anytime, select File > New > Custom > Initial Configuration Wizard.
