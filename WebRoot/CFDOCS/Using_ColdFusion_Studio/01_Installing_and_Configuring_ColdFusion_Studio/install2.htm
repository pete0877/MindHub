Chapter 1 :  Installing and Configuring ColdFusion Studio



Installing Studio

Follow the instructions below to install Studio from the ColdFusion Studio 4.5 CD. These instructions require you to restart your computer at the end of the installation. As a result, we recommend that you quit all other applications before you begin the installation.

To install ColdFusion Studio 4.5:
  1. Insert the Studio 4.5 CD in the CD-ROM drive. You see the Studio 4.5 Installation splash screen.
  2. Click Install.
  3. The InstallShield Wizard starts and you see the Welcome window.
  4. Click Next. You see the Software License Agreement.
  5. Click Yes to accept the Software License Agreement. You see the User Information window.
  6. Enter your name, company, and serial number information and click Next. You see the Choose Destination Location window.
  7. If you want to change the default destination, click the Browse button. Once you are satisfied with the destination, click Next. You see the Select Components window with the Studio Program Files and Documentation selected.
  8. Click Next. You see the Select Program Folder window with ColdFusion Studio 4.5 entered as the default program folder.
  9. Click Next. You see the Start Copying Files window.
  10. Click Next to begin copying files. When the copying is done, you see the Setup complete window.
  11. Select a restart option and click Finish to complete the installation.
