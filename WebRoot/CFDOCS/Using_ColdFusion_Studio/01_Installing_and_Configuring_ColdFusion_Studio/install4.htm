Configuring Browsers

Studio makes it easy to view HTML documents and to process CFML application pages.

Internal browser options

Open the Settings > Browse dialog (F8) to select a browser for viewing documents in Studio:

  • If Internet Explorer is detected on your system, you can use it as the internal browser.

    Go to http://www.microsoft.com/downloads/ to get the latest version.

  • If Netscape's Mozilla NGLayout control is detected, you can choose it.

    Go to http://www.mozilla.org/newlayout/for information on this open source browser development project.

  • A default internal browser is available if you wish to use it.

External browser options

Studio looks for a number of common browsers on your system and generates a list of the ones it finds. Click Options > Configure External Browsers to review and edit the list. You can add and delete browsers and edit their current settings.

This dialog also offers you choices for how Studio processes documents when you launch an external browser.
