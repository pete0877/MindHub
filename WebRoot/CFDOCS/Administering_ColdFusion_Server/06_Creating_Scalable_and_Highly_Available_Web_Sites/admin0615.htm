Chapter 6 :  Creating Scalable and Highly Available Web Sites



Other Informational Resources

This chapter covers a lot of information about creating scalable and highly available Web sites, and it touches the surface of many important topics that affect your Web applications' ability to scale and perform. To learn more about these and other related topics in greater detail, consult additional resources.

The following list of technology white papers, books, and other Web sites will get you started.

White papers

The following white papers provide a good balance between introductory and technology-specific information:

  • Allaire Corporation. "Scalable Web Application Development with ColdFusion," January, 1999. Acrobat PDF document available for download on the Allaire Web site at http://www.Allaire.com/developer/index.cfm. Click on the White Papers link at the bottom of the page.
  • Collaborative Research, Inc., Peter Christy and John Katsaros. "Internet Traffic Management Products," 1999. Acrobat PDF document available for download on the ITMCenter Web site at http://www.itmcenter.com/PDF/ITMwhitepaper.PDF.
  • Microsoft Corporation. "Microsoft Clustering Architecture." Available on the Microsoft Web site at http://www.microsoft.com/ntserver/ntserverenterprise/techdetails/prodarch/clustarchit.asp. Also see the following URL for good overview information about clustering: http://www.microsoft.com/ntserver/ntserverenterprise/exec/overview/clustering.asp
  • Cisco Systems. "Cisco DistributedDirector" and "The Effects of Distributing Load Randomly to Servers," April, 1999. Available for download on the Cisco Web site at http://www.cisco.com/warp/public/cc/cisco/mkt/scale/distr/tech/index.shtml.
