CFOBJECT

The CFOBJECT tag allows you to call methods in COM, CORBA, and JAVA objects.

Note ColdFusion administrators can disable the CFOBJECT tag in the ColdFusion Administrator Basic Security page.

On UNIX, COM objects are not currently supported by CFOBJECT.

CFOBJECT topics

  • CFOBJECT Type="COM"
  • CFOBJECT Type="CORBA"
  • CFOBJECT Type="JAVA"
