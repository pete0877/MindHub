LCase

Returns string converted to lowercase.

See also UCase.

Syntax

LCase(string)
string

String being converted to lowercase.

Examples

    <P>Your text, <CFOUTPUT>#FORM.sampleText#</CFOUTPUT>,
    returned in lowercase is  <CFOUTPUT>#LCase(FORM.sampleText)#

    <P>Please enter some text.
