ArrayIsEmpty

Determines whether the specified array is empty of data.

Returns a Boolean TRUE if specified array is empty, FALSE if not empty.

See also ArrayLen.

Syntax

ArrayIsEmpty(array)
array

Name of the array you want to check for data.

Example
