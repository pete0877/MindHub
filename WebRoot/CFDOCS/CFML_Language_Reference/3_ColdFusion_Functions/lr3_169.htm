LSIsDate

Like the IsDate function, LSIsDate returns TRUE if string can be converted to a date/time value in the current locale, FALSE otherwise.

Syntax

LSIsDate(string)
string

Any string value.

Usage

Years less than 100 are interpreted as 20th century values.
