Tan

Returns the tangent of a given angle.

See also Atn, ASin, Cos, Sin, and Pi.

Syntax

Tan(number)
number

Angle in radians for which you want the tangent. If the angle is in degrees, multiply it by PI()/180 to convert it to radians.
