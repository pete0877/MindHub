Chapter 2
ColdFusion Functions

This chapter describes each of the functions in the ColdFusion Markup Language (CFML). The introduction contains an alphabetical summary of ColdFusion functions, a list of new functions in ColdFusion 4.5, and a list of functions by category. The remainder of this chapter provides complete descriptions of each function, listed alphabetically.
