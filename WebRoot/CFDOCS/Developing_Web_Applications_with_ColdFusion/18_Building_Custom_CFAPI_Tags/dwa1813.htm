Implementing Java CFX Tags

Implementing a Java CFX requires interaction with the Request and Response objects passed to the processRequest method. In addition, CFXs that need to work with ColdFusion queries will also interface with the Query object. See the "Java CFX Reference" for a detailed description of CFX Java object types.
